  - search-service - Search functionality

  Infrastructure Components

  Database & Messaging:
  - PostgreSQL (5432) - Primary database

  - Redis (6379) - Caching

  Monitoring & Logging (ELK Stack):
  - Elasticsearch (9200) - Log storage
  - Logstash (5000) - Log processing
  - Kibana (5601) - Log visualization
  - Prometheus (9090) - Metrics collection
  - Grafana (3000) - Metrics visualization

  Security & Tracing:
  - Keycloak (8080) - Authentication/authorization
  - Jaeger (16686) - Distributed tracing

  Service Communication Flow

  The order simulation demonstrates a complete service chain:
  1. API Gateway ‚Üí routes /api/v1/order/simulate to inventory-service
  2. Inventory Service ‚Üí calls product-service
  3. Product Service ‚Üí calls ecom-order-service
  4. Order Service ‚Üí calls payment-service
  5. Payment Service ‚Üí calls notification-service

  Architecture Patterns

  - Service Discovery: All services register with Eureka
  - Load Balancing: Using Netflix Ribbon (lb://service-name)
